GeoSPARQL standard

  1. Core
  2. Topology Vocabulary: relations between geometries
    • Simple Features
    • Egenhofer
    • RCC8
  3. Geometry
    • GML
    • WKT
  4. Geometry Topology: properties of geometries & geometry functions
  5. RDFS Entailment: shape hierarchy
  6. Query Rewrite: rules for shapes (RIF)
